use_referrer_param_name(new_name) click to toggle source

Used to set a different param name where the referrer path should be stored. Default is :referrer, a sensible alternative would be :r or so. Example:

class MyController
  use_referrer_param_name :r
  # ...
end
# File lib/redirectr.rb, line 27
def use_referrer_param_name(new_name)
  const_set Redirectr::REFERRER_PARAM_NAME, new_name.to_sym
end
